## 个人成长记录

学习自我管理的大致可以分为4大类:
### 1. 不知不觉
听到看到了，觉得没啥用，不值得去做。
### 2. 后知后觉
了解了几个点，恍然大悟，立竿见影，取得了一些成果，以为自己懂了掌握了；
### 3. 当知当觉
认识到自我管理的必要和价值，看了很多书，上了很多课，却始终感觉无法融入自己的体系，无法找到更好的套路，无法长期应用；
### 4. 先知先觉
掌握了它的精髓，更新固化自己系统。
这里没有答案，只有套路方法，需要自己学习方法建立适合自己的系统去寻找答案。提高你做决策的能力，提升你应对风险的能力。
